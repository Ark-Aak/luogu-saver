## Highlights

- Built with Node.js 22
- Frontend: Nunjucks + SemanticUI
- Backend: Express + MySQL

### Getting Started

```bash
git clone https://github.com/luogu-saver-dev/luogu-saver.git
cd luogu-saver
npm install
node app.js
```
