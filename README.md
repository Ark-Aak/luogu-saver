洛谷保存站开源代码。

## 特性

- 基于 NodeJS 22 实现
- 前端采用 Nunjucks + SemanticUI
- 后端使用 Express + MySQL

### 使用

```bash
git clone https://www.github.com/laikit-dev/luogu-saver.git
cd luogu-saver
npm install
node app.js
```
